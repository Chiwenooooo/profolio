客服系統重構

UX / UI / 切版
專案背景 -

CTI 客服電話進線系統重構，是更新公司內部使用20年的客服電話進線系統，以前舊系統的觀念 "所見即所得"，因此將大部分的按鈕放都擺放在第一層的介面，客服人員反而在找尋功能、記錄案件時花更多的時間。在客服系統重構專案中負責 UX 使用者行為調查、UI 設計、網頁切版。

介面設計目標 -

「八二法則」利用簡潔的介面增加客服人員的效率，用20%的時間處理大多數基本問題，80％的時間可以處理更複雜的客服案件。改版後的成效顯示，平均每個案件減少了大約8%的案件通話時間。

Tools -
Adobe XD . Zeplin . HTML . CSS . JQuery
