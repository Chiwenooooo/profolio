由六角學院與Adobe XD Taiwan社群所舉辦的 - 前端精神精神時光屋，
主辦單位提供User story，UI設計師產出mock up後，前端工程師接續撰寫網頁程式。
在這個side project中擔任前端工程師的角色，藉由實作練習，精進前端的各項能力。

「ＷhiteSpace 旅館預約服務」首頁能看到所有房型，並能點擊任一房型進入到單一房型頁面觀看更多。在單一房型內頁，客戶可以觀看該房型詳細資訊，例如旅館描述、平日價格(一~四)、假日價格(五~日)、check in 時間、其他服務。在客戶填寫姓名、電話、預約起訖日後，可以看到房間總價格並進行線上訂房。

    RestFul API
  • Get 取得所有房型 / 單一房型細節
  • Post 預約房型
  • Delete 清除所有預約
  • Postman 串接資料時用來測試API的工具
    GitHub pages
  • Source tree 利用版控工具將網頁發布到GitHub pages
  • git 修改後陸續將程式更新和發布
    Plugin
  • moment.js 日期套件應用
