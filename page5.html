群益英雄聯盟是針對 "增加新客" 舉辦的行銷活動，
客戶可以藉由開戶、入金、推薦好友獲得商品卡，增加註冊開戶的意願。
在此專案主要負責網頁切版、資料串接。

    RestFul API
  • 參賽資格 判斷兩項活動的資格，各別顯示相對應的畫面
  • 推薦碼 利用身分證加上GTM來源別產生推薦碼，後續客戶可以將推薦碼分享至FB或LINE
  • 查詢戰績 串接兩項活動達成表
    CSS
  • clip-path 運用遮罩將主角斜切為三個陣營
  • animation.css 主視覺上簡易的動畫效果
    Plugin
  • swiper.js 輪播圖插件應用
