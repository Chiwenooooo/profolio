專案背景 -

"出發吧！群益聯盟" 是針對新客戶舉辦的行銷活動，藉由開立新帳戶、下單信用交易、推薦好友可獲得超商禮券，提升用戶開立帳戶的意願。在此專案主要負責資料串接、網頁切版。

專案說明 -
    RestFul API
  • 參加資格 判斷是否符合信用交易、推薦好友的活動資格，顯示相對應的活動達成表或推薦碼。
  • 活動達成表 串接用戶在各平台的下單完成狀態、各個活動的達成日期。
  • 推薦碼 利用身分證加密和判別 GA UTM（Urchin Tracking Module）廣告商來源所組成，新朋友輸入推薦碼後，便可以解密出推薦人並取得推薦人的流量來源。
  • 社群分享 動態產生含有用戶推薦碼的社群連結，提供客戶快速分享至 FB 或 LINE。
    網頁切版
  • CSS 運用 clip-path 遮罩將主視覺的主角斜切為三個陣營
  • Plugin swiper.js 輪播圖插件應用
Tools -
HTML5 . CSS . JavaScript . RESTful API
