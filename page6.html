專案背景 -

"行動贏家" 是一款公司內部自行開發證券下單產品，產品主要強調智能下單、科技化的特性，配合視覺設計師所設計出的介面，再運用微動畫強化科技、動態的感受。

專案說明 -

由於網頁的主視覺動畫不需要鼠標的互動，加上 JavaScript 所耗的資源較多，CSS animation 即使在系統負載超過 50% 仍可有效運作，因此選擇用 CSS animation 來製作動畫。

    CSS animation
  • Loading 預載入動畫避免畫面延遲，並帶出後面的產品主視覺
  • Key vision 將主視覺切割為前、中、後三段的手刻動畫
  • Float button 利用偽元素做出的按鈕效果
    Plugin
  • AOS.css 捲軸滑到各個區段時的視覺滾動差異，提升頁面的動態感
Tools - 
HTML5 . SCSS . CSS animation . JQuery
